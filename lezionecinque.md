<h1>Lezione PvP – Battaglia per Gilneas (Battle for Gilneas)</h1>

<h2>0) Obiettivo della lezione</h2>
<p>Alla fine di questa lezione devi saper:</p>
<ul>
  <li>Vincere gestendo <strong>3 nodi fissi</strong> con rotazioni pulite e difese solide.</li>
  <li>Capire che Gilneas è un BG di <strong>controllo territoriale</strong>, non di inseguimenti.</li>
  <li>Leggere correttamente <strong>tempi di rientro, distanze e priorità</strong>.</li>
</ul>

<hr>

<h2>1) Identità del BG </h2>
<p><strong>Formato:</strong> 10v10.</p>
<p><strong>Meccanica chiave:</strong> controllo di <strong>3 basi</strong> che generano punti nel tempo.</p>

<h3>Regola fondamentale</h3>
<ul>
  <li><strong>2 basi</strong> = partita persa nel medio periodo.</li>
  <li><strong>3 basi</strong> = unica condizione di vittoria.</li>
</ul>

<p>Battaglia per Gilneas è un BG di <strong>efficienza</strong>: chi spreca meno tempo vince.</p>

<hr>

<h2>2) Mappa in parole (layout e logica)</h2>
<p>Mappa <strong>triangolare</strong> con tre nodi equidistanti.</p>

<h3>Le 3 basi</h3>
<ul>
  <li><strong>Faro (Lighthouse)</strong>: base elevata, ottima per difesa e controllo visuale.</li>
  <li><strong>Miniera (Mine)</strong>: base laterale con più vie di accesso.</li>
  <li><strong>Acquedotto (Waterworks)</strong>: base centrale, snodo di rotazioni.</li>
</ul>

<h3>Caratteristiche chiave</h3>
<ul>
  <li>Distanze brevi: le rotazioni sono rapide.</li>
  <li>Nessun nodo “inutile”: perderne uno è sempre critico.</li>
</ul>

<hr>

<h2>3) Condizione di vittoria (macro)</h2>

<h3>Piano standard</h3>
<ul>
  <li>Prendere <strong>2 basi iniziali</strong>.</li>
  <li>Contestare e stabilizzare la <strong>terza</strong>.</li>
  <li>Difendere senza overcommit.</li>
</ul>

<h3>Principio chiave</h3>
<p>Gilneas non si vince attaccando a caso, ma <strong>difendendo bene ciò che hai</strong>.</p>

<hr>

<h2>4) Ruoli fondamentali</h2>

<h3>4.1 Difensore fisso</h3>
<ul>
  <li>Uno per base.</li>
  <li>Gioca per tempo, CC e sopravvivenza.</li>
  <li>Chiama aiuto con anticipo.</li>
</ul>

<h3>4.2 Rotatori</h3>
<ul>
  <li>Si muovono tra le basi.</li>
  <li>Rispondono alle chiamate.</li>
  <li>Puniscono over-rotate nemici.</li>
</ul>

<h3>4.3 Pressione / Attacco</h3>
<ul>
  <li>Attacca la base più debole.</li>
  <li>Non chasea fuori nodo.</li>
</ul>

<hr>

<h2>5) Apertura (start)</h2>

<h3>Obiettivo dell’apertura</h3>
<ul>
  <li>Stabilire 2 basi sicure.</li>
  <li>Impostare subito le difese.</li>
  <li>Non perdere Waterworks gratis.</li>
</ul>

<h3>Apertura solida</h3>
<ul>
  <li>Dividere il team in modo equilibrato.</li>
  <li>Lasciare subito un difensore su ogni base presa.</li>
</ul>

<h3>Errore tipico</h3>
<p>Lasciare una base vuota “solo per un attimo”. A Gilneas, quell’attimo basta per perdere.</p>

<hr>

<h2>6) Mid game – Dove si decide la partita</h2>

<h3>6.1 Difesa corretta</h3>
<ul>
  <li>Non serve uccidere tutti.</li>
  <li>Serve impedire il cap.</li>
</ul>

<h3>6.2 Rotazioni intelligenti</h3>
<ul>
  <li>Ruota in 1–2, non in 4.</li>
  <li>Mai lasciare una base scoperta.</li>
</ul>

<h3>6.3 Attacco efficace</h3>
<ul>
  <li>Colpisci dove il nemico difende peggio.</li>
  <li>Arriva insieme, no trickle-in.</li>
</ul>

<hr>

<h2>7) Micro-gioco per base</h2>

<h3>7.1 Faro</h3>
<ul>
  <li>Ottimo per difesa grazie all’altezza.</li>
  <li>Usa knock e LOS.</li>
</ul>

<h3>7.2 Miniera</h3>
<ul>
  <li>Più ingressi, più difficile da difendere.</li>
  <li>Serve attenzione costante.</li>
</ul>

<h3>7.3 Acquedotto</h3>
<ul>
  <li>Cuore della mappa.</li>
  <li>Chi controlla qui decide le rotazioni.</li>
</ul>

<hr>

<h2>8) Errori da principiante (e correzione)</h2>
<ul>
  <li>Chase fuori nodo → base persa.</li>
  <li>Over-difendere → altra base scoperta.</li>
  <li>Entrare da soli su una base → wipe inutile.</li>
  <li>Non chiamare aiuto in tempo.</li>
</ul>

<hr>

<h2>9) Shotcalling efficace</h2>

<h3>Call utili</h3>
<ul>
  <li>“Inc 2 Faro, tengo.”</li>
  <li>“Serve 1 Miniera ora.”</li>
  <li>“Acquedotto stabile, non muovete.”</li>
</ul>

<h3>Call inutili</h3>
<ul>
  <li>“Aiuto!” senza numero.</li>
  <li>“Fight qui!” senza nodo.</li>
</ul>

<hr>

<h2>10) Esercizi pratici</h2>

<h3>Esercizio A – Lettura mappa</h3>
<p>Ogni 20–30 secondi chiediti quale base è più vulnerabile.</p>

<h3>Esercizio B – Disciplina difensiva</h3>
<p>Se difendi, il tuo compito è guadagnare tempo, non fare kill.</p>

<h3>Esercizio C – Rotazioni minime</h3>
<p>Allenati a ruotare con il minimo numero di giocatori.</p>

<hr>

<h2>11) Checklist finale</h2>
<ul>
  <li>3 basi = vittoria.</li>
  <li>Difesa prima dell’attacco.</li>
  <li>Rotazioni piccole e precise.</li>
  <li>Mai lasciare un nodo vuoto.</li>
  <li>Gilneas si vince giocando pulito.</li>
</ul>
